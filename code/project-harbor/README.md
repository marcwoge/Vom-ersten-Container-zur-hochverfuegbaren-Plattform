# Project Harbor

Das durchgehende Praxisprojekt des Buches – Stufe für Stufe vom einzelnen
Container zur produktionsnahen Plattform.

| Stufe | Inhalt | Kapitel |
|------|--------|---------|
| 01-basic-web | Statischer Webserver | 5–7 |
| 02-web-db | Web + PostgreSQL + Adminer | 16–18 |
| 03-reverse-proxy | Traefik + TLS + Segmentierung | 19–23 |
| 04-monitoring | Prometheus + Grafana + cAdvisor | 29 |
| 05-logging | Loki + Grafana | 30 |
| 06-backup | Backup-/Restore-Skripte | 12, 37 |
| 07-load-balancing | Mehrere Instanzen + Proxy | 33–34 |
| 08-swarm | Swarm-Stack | 35 |

> Passwörter sind Platzhalter. `cp .env.example .env` und Werte ersetzen.
